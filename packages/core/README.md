# @nulab/datadog-ai-guard-middleware-core

Internal shared core package for [`@nulab/datadog-ai-guard-middleware-ai-v2`](https://www.npmjs.com/package/@nulab/datadog-ai-guard-middleware-ai-v2) and [`@nulab/datadog-ai-guard-middleware-ai-v3`](https://www.npmjs.com/package/@nulab/datadog-ai-guard-middleware-ai-v3).

> **This package is not intended for direct use.** All public APIs are re-exported through the middleware packages. Please use `@nulab/datadog-ai-guard-middleware-ai-v2` or `@nulab/datadog-ai-guard-middleware-ai-v3` instead.

## Why is this published?

This package is published to npm solely for dependency resolution. The middleware packages (`ai-v2` and `ai-v3`) depend on it at runtime. It does not provide any additional APIs beyond what the middleware packages already expose.
